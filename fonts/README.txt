LATIN CT (playbill font) — used only for the hero name "Eric Kofman".

Latin CT is a paid font from CastleType (myfonts.com). It is NOT included here
and cannot be redistributed. To make it appear on your live website:

1. Buy a WEBFONT license for the width you want (e.g. "Latin CT Not Wide" or
   "Semi Wide") from MyFonts. A Desktop license does NOT permit web hosting.
   - Individual style ~$59, full 6-width family ~$199 (desktop prices).
   - Web use is a separate, annual license tier.
2. You'll receive a webfont kit. Put the files in THIS folder and rename them:
       fonts/LatinCT.woff2
       fonts/LatinCT.woff
   (The @font-face rule in css/styles.css already points here.)
3. Reload the page — the hero name switches to Latin CT automatically.

Note: if you've installed the Desktop version on your own Mac/PC, the name will
already show in Latin CT when YOU open index.html locally, because the browser
can use system-installed fonts. Visitors, though, need the webfont kit above.

Until the files are added, the hero name falls back to Cinzel (a free
engraved-serif in a similar theatrical spirit).
